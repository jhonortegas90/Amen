# Lightwell Design Direction

The selected Amen visual direction is `Lightwell`: black glass surfaces, warm amber illumination, soft vertical depth, and a restrained prayer wall that keeps attention on anonymous intentions.

- Reference image: `docs/design/lightwell-reference.png`
- Primary background: `#0B0E14`
- Primary text: `#F5F5F5`
- Amen/pinned glow: `#FFD700`
- Motion: soft fades, slow ripple loops, tactile scale-down press, warm glow release.
- Product rule: no names, no avatars, no comments, no social graph.
